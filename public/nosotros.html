Somos desarrolladores web Full Stack formados en la Fundación Pescar, donde hemos adquirido habilidades tanto en el frontend como en el backend. Este es nuestro proyecto final, resultado de meses de aprendizaje y trabajo en equipo. La aplicación fue construida utilizando tecnologías clave como HTML5, CSS3, JavaScript, React para la gestión del frontend dinámico, y NodeJS como Backend junto a MongoDB como base de datos no relacional para manejar la información de manera eficiente.

Nuestro objetivo fue crear una plataforma intuitiva, rápida y escalable, aplicando principios de diseño responsivo y asegurando una experiencia de usuario fluida en cualquier dispositivo. Además, integramos buenas prácticas de desarrollo como la modularidad, reutilización de componentes, y control de versiones con Git.
